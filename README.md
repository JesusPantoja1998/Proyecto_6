# Proyecto\_6

Este repositorio contiene un analisis de una empresa que quiere hacer una campaña publicitaria acerca de videojuegos en base a datos estadisticos de años anteriores.

\*En primera instancia se limpian los datos para que no haya ningún tipo de problema con ellos al momento de usarse para realizar un calculoo grafica.



\*Después se calculan datos estadisticos que nos servirán como indicadores para saber si la campaña publicitaria será buena o mala.



\*por ultimo mostramos los datos obtenidos de manera grafica para un mejor entendimiento de las personas encargas de la campaña publicitaria.
